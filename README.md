# json Server
### Installation
* NodeJS: https://nodejs.org/en/
* IDE/Code Editor: VS Code: https://code.visualstudio.com/

```
npm install -g json-server
```

### Start JSON Server

```
* > cd db.json dir
```

```
json-server --watch db.json --port 5000
```

### Now if you go to 

```
* http://localhost:5000/
```
